1. На тестовой Базе Данных, аппроксимирущей работу магазина доставки продуктов, изучил следующие запросы, функции, операторы:
- SELECT и FROM: базовые операторы, которые позволяют выполнять запросы к БД;
- ORDER BY: оператор сортировки столбцов;
- LIMIT: ограничение количества строк на вывод;
- CAST или "::": преобразование типов данных строки к нужным;
- DATE_PART: выделение часов, минут, лет и тд из data types;
- COALESCE: для обработки NULL значений;
- CONCAT: для конкатенации строкового типа данных или тех, которые можно к нему привести;
- Различные математические функции: https://www.postgresqltutorial.com/postgresql-math-functions/;
- Арифметические операторы: https://www.postgresql.org/docs/9.3/functions-math.html;
- CASE, WHEN, THEN и END: составление комплексных логических выражений;
- AS: для переименования или создания имени нового столбца;
- Функции для работы со строками.
