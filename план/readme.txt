Краткая формулировка задачи на семестр:
Сервис, который на основе желаемой профессии выдает списком курсов, которые для этой профессии наиболее полезны.

Результат: готовая траектория. По базе данных алгоритм составляет ИОТ для студента.

Моя задача в данном проекте:
  Написать следующий алгоритм:
    Для данной профессии по БД алгоритм выбирает список полезных курсов. Далее из учебных дисциплин: какую можно собрать комбинацию курсов на основе того, что эти курсы полезные.
    Сперва курсы для данной профессии расставить по приоритету / дать курсам веса. Когда для данной дисциплины нужно будет порекомендовать список курсов, алгоритм рекомендует курс с наибольшим весом
